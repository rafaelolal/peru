Sobre EL Proyecto

Este es el proyecto final de español de Rafael Almeida y Edward Mendel, de la clase de la Sra. Vergara, A-7/8 2021-2022 en Union County Vocational Technical Schools

Estrategia

Rafael fue el responsable de crear todo el sitio web y parte de la traducción, y Edward de la investigación.

Tecnología

Este proyecto fue creado principalmente con Django, un framework de python, bootstrap, jquery y ajax. Este fue el primer proyecto que Rafael hizo usando Django y ajax para crear páginas responsivas que pueden actualizar la información mostrada sin refrescar la página.
